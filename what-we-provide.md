# What we provide

We provide offensive security testing and practical security assessments to help you find and fix real risks. We maximize our efficiency (thereby reducing your costs) by following an approach between grey and white box with a reasonable time box.

We focus on the following services:

* Web application pentests
* Active Directory / Entra ID pentests
* SOC / SIEM attack detection tests
* Infrastructure pentests
* Active Directory password analysis
* Web application load tests

## What we don't provide

We are experts in offensive security. Having the best knowledge in a domain makes focus mandatory.

This is why we don't provide:

* SOC/SIEM services
* Incident response
* ISMS consulting
* CISO as a service
* Awareness trainings

## Further Projects

We are also developing innovative projects that take our pentests to the next level.
